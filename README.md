# Samriddhi
Samriddhi is an AI-powered financial intelligence platform built for rural dairy micro enterprises like SHGs, FPOs, and individual farmers, who lack formal credit histories but operate real, data-rich businesses.

It uses a regression model trained on this data to predict enterprise cash flows 3–6 months into the future, and generates a 0–100 risk score for each enterprise using a hybrid rule-based + ML scoring engine.
